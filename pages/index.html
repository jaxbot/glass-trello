Press the button below to add the gtop instance of this server to your Glass. Uninstall it in My Glass.

gtop is a Glassware application that brings server monitoring and administration to your Google Glass device.

gtop is developed by Jonathan Warner, aka Jaxbot, and is provided as free, open source software.
You can find the source on Github, along with install instructions.
